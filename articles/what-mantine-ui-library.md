## Mantine とは、Mantine の特徴

すでに述べているとおり、OSS の React 専用無料 UI ライブラリです。

### メリット

- コンポーネントがとても豊富
  - DatePicker
- Tailwind CSS との相性も良い
- form の hook が提供されている
- デフォルトの UI に強くロックインされない
