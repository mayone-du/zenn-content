## 本文

### grid

### grid-cols-{n}

n 個のカラム(横の要素数)をもつグリッドかを指定。`grid`と一緒に、親要素に指定。

### gap-{n}

n スペース分の余白を要素と要素の間にもたせるか。親要素に指定。

### col-span

要素自体の幅。子要素に指定。
